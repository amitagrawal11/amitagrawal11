 ### 🤖 AI-Powered Loan Evaluation Assistant (Fintech)

**Problem**  
Loan evaluation workflows were manual, time-consuming, and error-prone — analysts had to interpret complex financial data across multiple systems.

**Solution**  
Designed and integrated an **AI-assisted workflow layer** using OpenAI APIs to guide users through loan evaluation steps.

**What I Built**
- AI-assisted input suggestions for financial parameters  
- Context-aware prompts for faster decision-making  
- Integration with existing React + GraphQL system  
- UI layer for real-time AI feedback inside workflows  

**Architecture**
- React + TypeScript frontend  
- Node.js middleware for prompt orchestration  
- OpenAI API integration  
- Stateful UI with Redux  

**Impact**
- ⏱ Reduced manual effort in evaluation workflows  
- ⚡ Faster decision-making for analysts  
- 🧠 Improved accuracy via guided inputs  
- 🚀 Early adoption of AI in core fintech workflows  

**Why it matters**
→ Demonstrates ability to integrate AI into **real production systems**, not just prototypes
