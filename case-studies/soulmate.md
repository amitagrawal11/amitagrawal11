# ❤️ Soulmate — Rethinking Modern Matchmaking

## The Story

While exploring popular matrimony and dating platforms, I noticed the same frustrating experience everywhere.

Users repeatedly encountered inactive profiles with no indication of whether members were genuinely active or had abandoned the platform months ago. Many users waited for responses that never arrived because stale profiles continued to be recycled.

I believed technology could solve this.

Instead of building another dating app, I designed **Soulmate**—a cross-platform social and matrimony platform focused on connecting users with active, nearby, and genuinely compatible people.

---

## What I Built

- **Smart Profile Freshness** — Highlighted new, recently active, stale, and online/offline profiles.
- **Nearby Discovery** — Integrated Google Maps to discover profiles within configurable radiuses (10–50 km).
- **Smart Match Score** — Compared user profiles, generated compatibility scores, and explained matching and mismatching areas.
- **Event & Café Planner** — Enabled matched users to schedule meetings through an invitation and approval workflow.
- **Real-time Chat** — Secure one-to-one messaging powered by Firebase.
- **Swipe Discovery** — Modern swipe-based profile exploration for faster browsing.

## Challenges Solved

- Eliminated stale profile discovery using profile freshness indicators.
- Built map-based nearby profile discovery with configurable search radius.
- Designed an intelligent compatibility scoring system with explainable results.
- Created an event planning workflow for real-world interactions.
- Implemented secure real-time messaging using Firebase.
- Balanced scalability while operating within Firebase's free-tier limits.

---

## Outcome

Soulmate remains a **private beta**. Operating a real-time social platform requires continuous cloud infrastructure, storage, and messaging resources. As an independent project, I chose to keep it private rather than release an under-supported product.

Today, a small group of friends actively use the application within Firebase's free-tier limits, providing valuable feedback that continues to shape the product.
